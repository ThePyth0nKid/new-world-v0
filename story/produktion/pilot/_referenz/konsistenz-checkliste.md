## Pruef-Workflow

### Vor der Generierung

1. Shot-Nummer im Drehbuch identifizieren
2. Cross-Referenz-Tabelle pruefen: Welche Checklisten gelten?
3. Relevante DNA-Blocks aus `charakter-blocks.md` kopieren
4. Stil-Codes aus `stil-codes.md` anwenden
5. Prompt zusammenbauen

### Nach der Generierung

1. Generiertes Bild/Video gegen JEDE relevante Checkliste pruefen
2. Bei Fehler: Prompt anpassen und neu generieren
3. Besonders pruefen:
   - Ios Stift-Position (wechselt je nach Shot!)
   - Lichtquelle (NIEMALS Sonnenlicht)
   - Screen-Darstellung (NIEMALS schwebend/holografisch)
   - Kids Ring (muss am RECHTEN Ringfinger sein)

### Haeufige Fehler

| Fehler | Korrektur |
|--------|-----------|
| Sonnenlicht in Hafen-Szene | Negativ-Tag `sunlight` + `natural lighting` hinzufuegen |
| Holografische Displays | Negativ-Tag `holographic, floating UI` verstaerken |
| Ios Stift fehlt | `ballpoint pen` explizit im Prompt + Position angeben |
| Kid wirkt zu jung | `mature, early 40s, crow's feet, silver temples` betonen |
| New World sieht steril aus | `worn textures, visible weld seams, different alloys` betonen |
| Io zu muskuloes | `lean wiry build, thin` betonen + Negativ `muscular` |
| Zu helle Farben | `muted color palette` verstaerken + Negativ `bright saturated colors` |
