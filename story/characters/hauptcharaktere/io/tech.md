# Io — Tech-Seele

## Kein fester Tech-Stack

Io repraesentiert keinen bestimmten Stack. Er ist der **User**. Der Mensch vor dem Bildschirm. Der Anfaenger, der lernt.

## Was er darstellt
- **Der Lernende:** Jeder Leser, der zum ersten Mal ein Terminal oeffnet
- **Der Prompter:** Er stellt Fragen — an die Crew, an die New World, an sich selbst
- **Der Integrator:** Am Ende verbindet er alles. Er wird der Full-Stack-Mensch.

## Seine Reise durch den Stack
| Akt | Was er lernt | Durch wen |
|-----|-------------|-----------|
| Akt 1 | Was ist dieses Schiff ueberhaupt? | Kid, Draco |
| Akt 2 | Jede Gruppierung, jedes System | Alle Crew-Mitglieder |
| Akt 3 | Dass alles zusammengehoert | Die Ultra Nova |

## Code-Metapher
```
// Io ist der Prompt
const io = {
  input: "Was ist das hier?",
  context: undefined,  // noch leer
  learning: true
}

// Am Ende der Reise:
const io_final = {
  input: "Was brauchen wir?",
  context: fullStack,  // alles gelernt
  learning: true       // immer noch. Regel 9.
}
```
