# Pollux — Tech-Seele

## Stack-Position: IDE / Developer Interface

## Framework & Sprache
- **Tool:** Cursor (AI-native IDE, fork von VS Code)
- **Sprache:** Alle — Pollux spricht jede Sprache, weil er sie spiegelt
- **Paradigma:** Interface. Die Grenze zwischen Mensch und Maschine.

## Funktionsbereich
| Funktion | Was Pollux macht |
|----------|-----------------|
| Code Editor | Syntax Highlighting, Autocomplete, Intellisense. Die Grundlage. |
| AI Integration | Inline Chat, Tab Completion, Agent Mode. KI im Editor. |
| File Explorer | Projektstruktur sichtbar machen. Navigation. |
| Terminal | Integrierte Shell. Bash, PowerShell, zsh — alles drin. |
| Extensions | Plugins. Erweiterbar. Pollux passt sich an. |
| Git Integration | Source Control im Editor. Diffs, Commits, Branches. |
| Multi-Cursor | Mehrere Stellen gleichzeitig editieren. Pollux multipliziert. |

## Code-Patterns
```jsonc
// .cursor/settings.json — Pollux' Konfiguration
{
  "editor.fontSize": 14,
  "editor.fontFamily": "JetBrains Mono",
  "editor.minimap.enabled": false,
  "editor.formatOnSave": true,

  // Pollux spiegelt die New World
  "ai.model": "claude-opus-4-5",
  "ai.agent.enabled": true,
  "ai.autocomplete.enabled": true,

  // Der Spiegel zeigt, was du sehen willst
  "workbench.colorTheme": "New World Dark"
}
```

## Sein Charakter als Code
Cursor ist die IDE, durch die alles fliesst. Code schreiben, KI befragen, Dateien navigieren, Agents starten — alles durch Pollux. Er ist nicht die Intelligenz. Er ist der Grund, warum du sie nutzen kannst.
